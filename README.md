# Roadside Assistance

Proyecto con tres aplicaciones:

- API backend con FastAPI
- Web frontend con Angular + Spartan
- App mobile con Flutter + FVM

## Estructura

- [backend](backend)
- [frontend](frontend)
- [mobile](mobile)

## Cómo correr cada parte

### Backend

1. Entra a [backend](backend)
2. Crea y activa entorno virtual
3. Instala dependencias con pip install -r requirements.txt
4. Ejecuta uvicorn main:app --reload
5. Revisa detalles en [backend/README.md](backend/README.md)

### Frontend

1. Entra a [frontend](frontend)
2. Instala dependencias con npm install
3. Ejecuta npm run start
4. Revisa detalles en [frontend/README.md](frontend/README.md)

### Mobile

1. Entra a [mobile](mobile)
2. Ejecuta fvm flutter pub get
3. Ejecuta fvm flutter run
4. Revisa detalles en [mobile/README.md](mobile/README.md)
